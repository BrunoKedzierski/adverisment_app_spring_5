Welcome to the Stock Manager !

This is a simple Spring Boot 5 app. It is connected to a databes containing information about products, manufacturers and shops. The application provides basic CRUD operations, validation, image uploading and more! Techonologies used are:

  • Java 11
  • Spring 5 and Spring Boot
  • Hibernate
  • Lombok
  • Thymeleaf
  • Spring Security
  • Docker
  • Bootstrap
